# page-book-generator

Page Book Generator


## Install from PYPI

The homepage in pipy is https://pypi.org/project/page_book_generator/

```bash
pip install --upgrade page_book_generator
```

Using:

```bash
page-book-generator
```

## Install from source
Installing `page-book-generator` program

```bash
git clone https://github.com/trucomanx/PageBookGenerator.git
cd PageBookGenerator
pip install -r requirements.txt
cd src
python -m build
pip install dist/page_book_generator-*.tar.gz
```
Using:

```bash
page-book-generator
```

## Uninstall

```bash
pip uninstall page_book_generator
```
